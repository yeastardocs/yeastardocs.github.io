# 空格

* 中文与英文之间需要空格。

      **正确示例**：本手册适用于需要准备，配置和操作 PBX 系统的管理员。

      **错误示例**：本手册适用于需要准备，配置和操作PBX系统的管理员。

* 中文与数字之间需要空格。

      **正确示例**：同时注册数最多为 3。

      **错误示例**：同时注册数最多为3。

* 数字与单位之间不需要空格。

      **正确示例**：10TB

      **错误示例**：10 TB

* 引用链接前后需要空格。

      **正确示例**：更多信息，请参见 自动配置局域网中的 IP 话机 (PnP 方式) 和 自动配置 Yeastar TA FXS 网关 (PnP 方式)。
      
      **错误示例**：更多信息，请参见自动配置局域网中的 IP 话机 (PnP 方式) 和 自动配置 Yeastar TA FXS 网关 (PnP 方式)。
